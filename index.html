SICPer

我们因《SICP》相遇，SICP 这本书给我们展现了一全新世界——函数式编程。

共同的梦想使我们相聚，不懈的坚持让我们走的更远。 －－2015/12/15

定位

我们不仅仅是 SICP 读书小组，更准确的说，关于编程一切的东西都是我们讨论的话题。分享、交流，真正成为一名增长黑客。

计划

每周一次分享。主要包括两个方面：项目＋题目，项目偏向实践，题目偏向理论，一定要防止眼高手低，不管觉得多么 low 的项目或题目，做了才有发言权。

分享时有个主讲人，他会提前预告分享主题，其他人课前预习，带着问题听。同时主讲人负责写每周的总结，总结的形式是最终发个文章出来，发在我们的 github 仓库中，文章形式不限，可以是markdown、ppt等。分享一定要有自己原创的东西。

惩罚措施

红包。主讲人延迟一次：100 元，如果以后想退出：500 元。

说明：

  1. 如果主讲人分享的主题某个人不感兴趣，可以不参与，但是需要提前说明，并且一个月只有一次机会
  2. 多去“喷”，但喷之前需要多去思考，给出喷的具体理由以及改进措施
  3. 有任何问题随时在群里发问，发挥个人主动性
  4. 分享时间尽量安排在周四或周五，主讲人需要协调其他人与自己的时间
  5. 以上规定自 2015/12/15 日生效。
